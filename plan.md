## Contraintes techniques

- **Stack** : HTML5, CSS3, JavaScript vanilla — aucun framework
- **Hébergement cible** : GitHub Pages (fichiers statiques uniquement)
- **Responsive** : Mobile first, breakpoints à 768px et 1024px
- **Accessibilité** : Respecter les bases WCAG 2.1 (contrastes, balises aria, navigation clavier)
- **SEO** : Balises `<title>`, `<meta name="description">`, balises Open Graph de base
- **Performance** : Pas de dépendances externes inutiles, images optimisées
- **Formulaire de contact** :
- **Palette** : Sobre et professionnelle
