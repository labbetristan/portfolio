# Audit Communication & Marketing

## Ce qui fonctionne bien

- **Positionnement clair** : "TPE, artisans, associations" — la cible est identifiée dès le H1
- **Proposition de valeur** : "Vous restez concentré sur votre métier" — bonne accroche orientée bénéfice client
- **Processus transparent** : les 4 étapes (Écoute → Conception → Développement → Livraison) rassurent
- **Ton juste** : professionnel mais accessible, pas de jargon technique — cohérent avec la cible
- **CTAs multiples** : chaque page pousse vers le contact, bon tunnel de conversion
- **Mention "devis gratuit"** : lève le frein prix pour la cible (TPE/artisans)
- **Dark mode + accessibilité RGAA** : différenciant technique exploitable commercialement

## Suggestions d'amélioration

### 1. Preuve sociale insuffisante

**Problème** : "10+ ans d'expérience" et "100% à distance" sont les seuls chiffres. Pas de témoignages, pas de nombre de projets livrés.

**Suggestion** :
- Ajouter 2-3 **témoignages clients** (même courts) sur la page d'accueil
- Ajouter un compteur "projets livrés" (même si c'est 5-10, ça rassure)
- Sur chaque projet du portfolio, ajouter un mini-témoignage ou le résultat obtenu (ex : "Fréquentation doublée en 3 mois")

### 2. Portfolio trop maigre en contexte

**Problème** : 3 projets avec des descriptions très courtes (1 ligne). On ne comprend pas la valeur apportée.

**Suggestion** :
- Créer une **page détail par projet** avec : le brief client, les problèmes résolus, les choix techniques, le résultat/impact
- Ajouter des **tags de technologie** sur chaque carte (WordPress, Astro, etc.) pour montrer la polyvalence
- Indiquer le **type de client** (association, événement, recherche) pour que les prospects s'identifient

### 3. Pas de tarif indicatif

**Problème** : "Sur devis" et "Devis gratuit" sont rassurants mais pas suffisants. La cible (TPE/artisans) a besoin d'une idée de budget pour se projeter.

**Suggestion** :
- Ajouter des **fourchettes de prix** sur la page services : "Site vitrine : à partir de XXX €", "Refonte : à partir de XXX €"
- Ou un tableau comparatif des formules (basique / standard / premium)
- Ça filtre aussi les prospects non qualifiés et fait gagner du temps

### 4. Pas de blog / contenu SEO

**Problème** : le site est purement vitrine. Il n'y a aucun contenu éditorial pour le référencement long-traîne.

**Suggestion** :
- Ajouter une section **blog** avec des articles ciblés : "Combien coûte un site vitrine en 2026 ?", "Pourquoi refondre son site ?", "Site WordPress vs site sur-mesure"
- Ces articles attirent du trafic organique qualifié et positionnent l'expertise

### 5. Accroche hero trop générique

**Problème** : "Des sites web sur-mesure pour les TPE, artisans et associations" — c'est clair mais ça pourrait être dit par n'importe quel freelance.

**Suggestion** :
- Essayer une accroche plus **orientée résultat** : "Vos clients vous trouvent en ligne. Je m'occupe du reste." ou "Un site pro qui travaille pour vous — même quand vous dormez."
- Garder la cible dans le sous-titre plutôt que dans le H1

### 6. Page contact : pas de téléphone / calendrier

**Problème** : le seul moyen de contact est le formulaire. La cible (artisans, TPE) préfère souvent un appel.

**Suggestion** :
- Ajouter un **numéro de téléphone** ou un lien vers un **outil de prise de RDV** (Calendly, Cal.com)
- Mentionner un délai de réponse concret : "Je vous réponds sous 24h" plutôt que "rapidement"

### 7. Différenciation faible sur les services

**Problème** : "Site vitrine", "Refonte", "Évolution" — c'est descriptif mais pas différenciant. Ça ne dit pas pourquoi choisir ce prestataire plutôt qu'un autre.

**Suggestion** :
- Ajouter des **bénéfices concrets** par service : "Site vitrine optimisé SEO — soyez visible sur Google dès le lancement", "Refonte sans perte de référencement"
- Mentionner l'accessibilité RGAA comme argument commercial ("Un site accessible à tous vos visiteurs, y compris en situation de handicap")

### 8. LinkedIn seul en réseau social

**Problème** : un seul lien social (LinkedIn) dans le footer.

**Suggestion** :
- Si un GitHub public avec des projets existe, le lier (crédibilité technique)
- Considérer un profil Malt ou autre plateforme freelance pour renforcer la confiance

### 9. Pas de mention de garantie ou SAV

**Suggestion** :
- Ajouter une mention du type "Garantie 30 jours après livraison" ou "Support inclus pendant X mois"
- Ça lève considérablement le frein à l'engagement pour un artisan qui investit
